Simple AWS KMS encrypt / decrypt demo
=====================================

A simple AWS KMS encrypt / decrypt demo, based on Node.js / Typescript Starter template project, runs on localhost environment.

Requirements
------------

- Node.js
- Typescript
- AWS Free Tier account
